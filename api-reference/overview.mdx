https://api.dihub.com.br
```

```

O ambiente de homologação precisa ser liberado pela equipe da DiHub. A habilitação desse acesso pelo próprio usuário ainda está em desenvolvimento.
</Warning>

## Autenticação

Todas as rotas exigem o header `X-Api-Token`.

```bash
X-Api-Token: YOUR_API_TOKEN
```

<Card title="Como gerar seu token" icon="key" href="/introduction/authentication">
  Veja como criar, copiar e revogar tokens no dashboard.
</Card>

## Fluxo básico

1. Gere seu token de API.
2. Consulte `GET /credits/balance` para validar seu saldo.
3. Envie `POST /diligences/emit` para iniciar a diligência.
4. Acompanhe o processamento e baixe os arquivos quando estiverem prontos.

<Info>
Use a página [Consumo de créditos](/reference/consumo-creditos) para validar a cobrança antes de integrar fluxos em lote.
</Info>
