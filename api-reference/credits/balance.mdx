---
title: 'Consultar Saldo de Créditos'
description: 'Retorna o saldo total de créditos disponível para sua conta'
---

Permite que você consulte o saldo de créditos disponível em sua conta DiHub, incluindo créditos pessoais e da organização.

## Endpoint

```http
GET /credits/balance
```

## Autenticação

Esta rota requer autenticação via chave de API. Inclua seu token no header `X-Api-Token`.

## Resposta de Sucesso

A API retorna um objeto contendo o saldo total disponível:

```json
{
  "balance": 15.5
}
```

### Campos da Resposta

- `balance` (number): Saldo total de créditos disponíveis (soma de créditos pessoais + créditos da organização)

## Exemplo de Requisição

```bash
curl --location 'https://s87bfswpmi.execute-api.us-east-1.amazonaws.com/credits/balance' \
--header 'X-Api-Token: 01KE2M9KJMA3AHKB45WE72SJGX:MTFiMTIzODctNDkzNy00OTBiLTkxMDItZjU3Y'
```

## Possíveis Erros

### 401 - Não Autorizado

Retornado quando o token de API está ausente ou é inválido.

```json
{
  "error": "Unauthorized",
  "message": "Missing or invalid api token"
}
```

**Solução**: Verifique se você está enviando o header `X-Api-Token` com um token válido.

### 500 - Erro Interno

Retornado quando ocorre um erro no servidor.

```json
{
  "message": "Internal Server Error"
}
```

**Solução**: Tente novamente em alguns instantes. Se o problema persistir, entre em contato com o suporte.

## Casos de Uso

- **Verificar saldo antes de criar diligências**: Consulte o saldo disponível antes de emitir novas diligências
- **Monitorar consumo**: Acompanhe o consumo de créditos da sua aplicação
- **Alertas de saldo baixo**: Implemente alertas automáticos quando o saldo estiver baixo
- **Dashboard**: Exiba o saldo disponível no painel da sua aplicação

## Notas Importantes

- Esta rota **não consome créditos**
- O saldo retornado é a soma de créditos pessoais e da organização
- O saldo é atualizado em tempo real após cada operação
