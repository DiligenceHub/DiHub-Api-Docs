A DiHub utiliza um sistema de créditos para cobrar pelo uso da plataforma. Cada diligência emitida consome créditos conforme as regras abaixo.

## Tabela de consumo

| Cenário | Créditos |
|---------|----------|
| Todas as certidões oficiais | 1 crédito |
| Análise de DiHub | 1 crédito |
| Todas as certidões oficiais + Análise de DiHub | 1,5 crédito |
| Todas as certidões oficiais com múltiplas comarcas no <ins>mesmo estado</ins> | 1 crédito |
| Todas as certidões oficiais com múltiplas comarcas em <ins>estados diferentes</ins> | 2 créditos |
| Todas as certidões oficiais com múltiplas comarcas em <ins>estados diferentes</ins> + Análise de DiHub | 2,5 créditos |

## Resumo

- **Uma comarca ou múltiplas no mesmo estado**: 1 crédito
- **Múltiplas comarcas em estados diferentes**: +1 crédito adicional
- **Análise DiHub adicional**: +0,5 crédito quando combinada com certidões oficiais

## Observações

- A análise de DiHub só é realizada para titulares com 18 anos ou mais caso seja solicitada para um menor de idade o seu crédito não será cobrado.
- O saldo é verificado antes da emissão; em caso de créditos insuficientes, a API retorna erro com o saldo disponível.
