## Quando usar cada um

|                    | MCP Server                                  | llms.txt                                       |
| ------------------ | ------------------------------------------- | ---------------------------------------------- |
| **Objetivo**       | Consulta dinâmica da documentação via IA    | Fornecer toda a documentação como contexto     |
| **Ideal para**     | Assistentes contínuos (Claude, Cursor)      | Prompts pontuais ou ferramentas sem MCP        |
| **Formato**        | Servidor MCP consultado em runtime          | Arquivo de texto consumido pelo modelo         |
| **Configuração**   | URL do endpoint no cliente MCP              | Cole o conteúdo ou aponte a URL                |
